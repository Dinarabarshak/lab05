# Лабораторна робота №5
Циклічні конструкції (for, while, do-while)

## Завдання
Підвести число `n` у ступінь `m` (n^m), продемонструвавши однаковий результат для
циклів `for`, `while` та `do-while`.

## Реалізація
- Програма запитує початкове та кінцеве значення діапазону.
- Перебирає числа за допомогою циклу `for`.
- Зберігає останнє знайдене число, яке ділиться на 7.
- Виводить результат на екран.
- Програма зчитує основу `n` та ступінь `m` з консолі.
- Перевіряє, що ступінь є невід'ємним цілим числом.
- Обчислює результат трьома циклами (`for`, `while`, `do-while`) і звіряє їх
  узгодженість.
- Виводить результат для кожної реалізації.
