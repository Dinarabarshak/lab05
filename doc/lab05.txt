+Лабораторна робота №5. Циклічні конструкції (for, while, do-while)
+Завдання: підвести число n у ступінь m.
+
+Хід виконання завдання
+- Створення проєкту: каталог lab05 з підкаталогами src, doc, bin; основний файл src/main.c.
+- Точка входу: функція main у файлі src/main.c. Саме вона викликається середовищем
+  виконання після старту програми, тому точка входу єдина і визначає порядок роботи
+  алгоритму.
+- Запуск програми: збірка через `make`, далі виконуваний файл bin/app запускається
+  з консолі. Приклад: `./bin/app` із введенням параметрів n та m.
+- Використання циклів: ступінь обчислюється трьома варіантами (for, while, do-while),
+  щоб підтвердити однаковий результат незалежно від циклічної конструкції.
+- Налагодження: програма дозволяє зупинку на breakpoints усередині будь-якої функції
+  power_* або у тілі main. Під час зупинки можна змінити значення лічильника або
+  проміжного результату, щоб простежити вплив на підсумковий вивід.
